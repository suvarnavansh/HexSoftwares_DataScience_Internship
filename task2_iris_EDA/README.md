# 🌸 Task 2 – Iris Dataset Exploratory Data Analysis (EDA)

## 📌 Overview

This task focuses on performing **Exploratory Data Analysis (EDA)** on the Iris dataset to understand its structure, statistical properties, and feature relationships. The analysis includes checking for missing values, summarizing the dataset, and creating visualizations to explore the distribution of features.

The Iris dataset is one of the most commonly used datasets for practicing data analysis and machine learning techniques.

---

## 🎯 Objectives

* Load and explore the dataset structure
* Identify and handle missing values (if any)
* Generate statistical summaries of the dataset
* Visualize feature distributions using graphs and plots
* Understand relationships between different features

---

## 📂 Dataset Information

The dataset contains measurements of iris flowers belonging to three species.

### Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

### Total Samples

150 observations across 3 different species of iris flowers.

---

## 🛠 Tools and Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📊 Analysis Performed

The following steps were performed in this task:

1. Importing required libraries
2. Loading the dataset
3. Exploring dataset structure (`head()`, `shape`, `info()`)
4. Checking for missing values
5. Generating statistical summaries using `describe()`
6. Visualizing feature distributions using:

   * Histograms
   * Pairplots
   * Boxplots
   * Correlation heatmap

---

## 📈 Key Insights

* Petal length and petal width show strong correlation.
* The Setosa species is clearly separable from the other two species.
* Versicolor and Virginica species show some overlapping characteristics.
* No missing values were found in the dataset.
